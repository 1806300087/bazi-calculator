### 2. 五虎遁月诀

月干计算依据五虎遁月诀（见《三命通会》）：

```
甲己之年丙作首，
乙庚之岁戊为头，
丙辛必定寻庚起，
丁壬壬位顺行流，
若问戊癸何方发，
甲寅之上好追求。
```

| 年干 | 正月起 |
|------|--------|
| 甲、己 | 丙寅 |
| 乙、庚 | 戊寅 |
| 丙、辛 | 庚寅 |
| 丁、壬 | 壬寅 |
| 戊、癸 | 甲寅 |

### 3. 五鼠遁日诀

时干计算依据五鼠遁日诀（见《三命通会》）：

```
甲己还加甲，
乙庚丙作初，
丙辛从戊起，
丁壬庚子居，
戊癸何方发，
壬子是真途。
```

| 日干 | 子时起 |
|------|--------|
| 甲、己 | 甲子 |
| 乙、庚 | 丙子 |
| 丙、辛 | 戊子 |
| 丁、壬 | 庚子 |
| 戊、癸 | 壬子 |

### 4. 干支纪年法

- 基准：1900年为庚子年
- 天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸（10循环）
- 地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥（12循环）
- 干支组合：60年一甲子循环

### 5. 五行归属

**天干五行：**
- 木：甲（阳木）、乙（阴木）
- 火：丙（阳火）、丁（阴火）
- 土：戊（阳土）、己（阴土）
- 金：庚（阳金）、辛（阴金）
- 水：壬（阳水）、癸（阴水）

**地支五行：**
- 木：寅、卯
- 火：巳、午
- 土：辰、戌、丑、未
- 金：申、酉
- 水：子、亥

## 🔧 使用方法

1. 打开 `index.html` 文件
2. 输入出生日期
3. 选择出生时辰（按2小时一个时辰）
4. 选择性别
5. 点击"排盘计算"按钮

## ⚠️ 重要说明

### 计算精度

1. **年柱精度**：✅ **V2已修复** - 根据立春节气精确判断年份
   - 立春前：算上一年
   - 立春后：算当前年
   - 误差：±1天（节气计算本身的精度限制）

2. **月柱精度**：✅ **V2已改进** - 根据24节气划分月份
   - 使用节气表确定月份
   - 每月节气前后可能存在±1天误差

3. **日柱精度**：✅ 准确 - 使用基姆拉尔森公式（标准天文算法）

4. **时辰划分**：按传统24小时分为12个时辰，每个时辰2小时。需要注意：
   - 子时分为早子时（0:00-0:59）和夜子时（23:00-23:59）
   - 本工具统一按夜子时计算

3. **历法转换**：输入为公历日期，计算基于天文历法。农历转公历或涉及1900年以前日期时可能存在偏差。

### 数据真实性

- ✅ 所有藏干数据来自《渊海子平》
- ✅ 五虎遁、五鼠遁口诀来自《三命通会》
